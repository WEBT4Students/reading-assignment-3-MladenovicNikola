World Wide Web General

What are the most essential technologies that made the World Wide Web such a success?

HTTPS (Hypertext Transfer Protocol), URL (Uniform Resource Locator), HTML, SMTP (Simple Mail Transfer Protocol), FTP (File Transfer Protocol).

How are resources (html pages, images, etc) addressed in the web?

They are adressed with the URL (Uniform Resource Locator).

What are the general components of a URL?

There are 5 components => Scheme (https://), Subdomain (www), Second-level-Domain (goggle), Top-level-Domain (com), Subdirectory (/ index.html/).

Web Server

What is a web server?

It is computer software and underlying hardware, that accepts requests via HTTPS or its secure variant HTTPS.

What does a web server?

It uses HTTP and other protocols (SMTP, FTP) to respond to client requests made over the World Wide Web (WWW).

The Main Job of a web server is to display website content through storing, processing and delievering webpages to users.

How can content be stored on a web server?

They store data on a server using some kind of database (server-side-storage). Then run server-side code to retrieve needed data, insert

it into static page templates, and serve the resulting HTML to the client to be displayed by the users browser.

How is content in webpages described?

Page content refers to all the information contained in a website. Page content can be displayed as text, links, images, audio, animation

or videos among other things.

Web Client

What is a web client?

It is an application used for connecting to a web server over HTTP (web browser Firefox).

What does a web client?

It retrieves it's files from a web server to find the a website which suits the user request and to graphically render the page on the users srceen.

Web Client and Web Browser are they the same?

Yes, they are the same.

Declarations for font families

he font-family property specifies the font for an element.

The font-family property can hold several font names as a "fallback" system. If the browser does not support the first font, it tries the next font.

Two types of font family

family-name - The name of a font-family, like "times", "courier", "arial", etc.

generic-family - The name of a generic-family, like "serif", "sans-serif", "cursive", "fantasy"

Declarations for font style normal and italic

The font-style property is mostly used to specify italic text.

Three values

normal - The text is shown normally

italic - The text is shown in italics

oblique - The text is "leaning"

font-style: normal // font-style_ italic

Font Variant // The font-variant property specifies whether or not a text should be displayed in a small-caps font. (font-variant:normal // small-caps)

Declarations for font sizes

The font-size property sets the size of the text.

Absolute size:

Sets the text to a specified size

Does not allow a user to change the text size in all browsers (bad for accessibility reasons)

Absolute size is useful when the physical size of the output is known

Relative size:

Sets the size relative to surrounding elements

Allows a user to change the text size in browsers

Set Font Size With Pixels => font-size: 40 px;

Set Font Size With Em => font-size: 2.5em; (40 px / 16 = 2.5 em)

Use Combination Of Percent and EM => he solution that works in all browsers, is to set a default font-size in percent for the element:

Declarations for font weights

The font-weight property sets how thick or thin characters in text should be displayed.

The font-weight property specifies the weight of a font (font-weight:normal) (font-weigth:bold);

font-weight: normal; // font-weight: bold; // font-weight: 900;
